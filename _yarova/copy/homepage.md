## THE TWO TRACKS

**H2:** Pick the track that fits where you want to go.

---

**DevOps & DevSecOps**

What you do: build, automate, and secure the pipeline that ships software.

You own:
- CI/CD (Continuous Integration / Delivery — the automated pipeline that builds and ships code)
- Infrastructure as Code with Terraform (writing cloud setup as reusable scripts)
- Container security and image promotion

Stack: AWS · Terraform · GitHub Actions · Kubernetes · Docker

---

**Site Reliability Engineering (SRE)**

What you do: keep production systems alive, observable, and self-healing.

You own:
- Telemetry — metrics, logs, traces (the three ways to know what a system is doing)
- Incident response and on-call runbooks
- Error budgets (how much downtime is acceptable before you must act)

Stack: AWS · Kubernetes · Prometheus · Grafana · PagerDuty

---

**Shared foundation (both tracks):**

Before either track begins, the base infrastructure is built together.

You provision:
- Virtual networks (VPCs)
- Identity and access permissions (IAM)
- The Kubernetes cluster both tracks run on

Stack: AWS · Terraform

---

**Bottom line:**
Every ticket you complete is a real commit on your GitHub.
Hiring managers open it. They inspect it. It holds up.

## THE 90-DAY JOURNEY
